Privacy Policy for Gerrit Reviewer Helper

Data Collection

This extension does NOT collect, store, or transmit any personal data or user information.

Data Usage

The extension operates entirely locally within your browser:

  • Reviewer email addresses are stored locally in Chrome's sync storage
  • All processing happens locally on your machine
  • No data is sent to external servers
  • No analytics or tracking mechanisms are used

Permissions Explanation

The extension requires the following permissions:

  • activeTab: To interact with Gerrit review pages when activated
  • alarms: To keep the extension service worker active for reliable performance (does not collect any data)
  • downloads: To allow export of debug logs to your local machine
  • storage: To save your configured reviewer list locally in browser storage
  • host permissions: To run only on specified Gerrit code review sites

Alarms Permission Details

The alarms permission is used solely for:

  • Maintaining extension service worker activity
  • Ensuring reliable performance when the extension is used after periods of inactivity
  • Preventing automatic termination by the browser

Note: The alarms functionality does not collect user data, track usage, or communicate with external servers. It only creates internal timers to keep the extension ready for use.

Data Storage

All data is stored locally in your browser:

  • Reviewer email lists → Chrome sync storage
  • Debug logs → Chrome local storage (optional export only)
  • No data is transmitted to external servers
